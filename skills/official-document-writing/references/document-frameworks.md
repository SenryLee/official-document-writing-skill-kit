# 文种框架

用于确定具体文种结构。单位历史格式、上级通知要求、用户给定模板优先于通用框架。

## 目录

- 工作简报
- 工作计划
- 工作总结
- 工作报告
- 心得体会
- 领导讲话
- 工作通知
- 工作方案
- 述职报告
- 调研报告
- 理论文章
- 新闻稿
- 通讯稿
- 先进事迹
- 民主生活会对照检查材料
- 党课
- 微型党课 / 演讲
- 会议纪要

## 工作简报

适用：阶段性成绩、内部宣传、工作亮点、专项工作分支成果。

结构：

1. 标题：使用单短语标题或双短语标题，如“开展专项整治”或“强化组织领导，责任体系更加健全”。
2. 首段：时间状语 + 按 + 拿 + 推。
3. 主体：3-5 个分点，每个分点写 `措施 + 成绩`，突出产品化名词和数据。

避免：理论太多、背景太长、没有成绩数据、只写未来安排。

## 工作计划

适用：年度、月度、阶段性未来安排，也可作为总结、报告中的下一步打算。

结构：

1. 标题：`20XX年工作计划/工作要点`。
2. 首段：工作思路，通常使用 `按 + 拿 + 推`。
3. 主体：3 个以上分点，每点写 `计划 + 目标`。

规则：

- 基本使用措施短语，如“推进重点任务”“完善制度机制”“提升服务质效”。
- 少写精准数字，除非上级已明确指标。
- 先读上级计划、要点、方案，再转化为本级安排。

## 工作总结

适用：月度、季度、半年、年度、多年或专项工作回顾。

结构：

1. 标题：`XX工作总结` 或 `20XX年XX工作总结`。
2. 首段：时间状语 + 按 + 拿 + 推。
3. 一、工作完成情况：`措施 + 成绩`。
4. 二、存在的主要问题：视要求决定是否写。
5. 三、下一步工作打算：视要求决定是否写。

规则：

- 工作完成情况必写。
- 是否写问题和下一步，要看上级要求和单位习惯。
- 素材来自计划、方案、通知、简报、新闻、阶段小结和数据。

## 工作报告

适用：年度正式回顾、重大会议、较大单位或规范场合。

结构：

1. 标题：`20XX年工作报告`。
2. 首段：优先沿用单位惯例。
3. 一、上一阶段工作情况：
   - 整体完成情况。
   - 主要成绩和数据。
   - 主要做法。
   - 必要时表示感谢。
   - 存在问题。
4. 二、下一阶段工作安排：
   - 形势分析。
   - 工作目标。
   - 工作思路。
   - 主要任务。
5. 结尾：号召。

## 心得体会

适用：开会、培训、学习文件、参观、阅读后的认识和行动。

结构：

1. 标题：`关于XX的心得体会`。
2. 首段：活动背景 + 感触认识 + 引出下文。
3. 主体：3-5 个分点，每点 `水分 + 干货`。
4. 落脚点：回到本职工作和下一步行动。

规则：

- 不写学生读后感。
- 不能只谈感受，必须落到工作。

## 领导讲话

适用：领导在会议、活动、部署、总结、推进场景中的发言。

先分类型：

| 类型 | 适用 | 核心结构 |
|---|---|---|
| 事前类 | 工作启动前动员部署 | 思想意义 -> 核心工作 -> 对内要求 -> 号召 |
| 事中类 | 工作推进中再部署 | 成绩/问题 -> 核心工作 -> 对内要求 -> 号召 |
| 事后类 | 工作结束后总结提升 | 成绩/做法/问题 -> 巩固提升 -> 对内要求 -> 号召 |
| 特殊类 | 一次性特殊任务 | 按措施主题分组 -> 号召 |

通用结构：

1. 标题：使用单短语标题或双短语标题，如“凝聚思想共识”或“聚焦重点任务，推动落实见效”。
2. 称谓。
3. 首段：会议背景、前文衔接、讲话缘由。
4. 主体：按类型选择 2-4 个大点。
5. 结尾：号召。

## 工作通知

适用：上级对下级安排工作。

通用工作通知：

1. 标题：`XX单位关于XX的通知`。
2. 主送机关。
3. 首段：背景 + 目的 + 依据 + `现将有关事项通知如下`。
4. 主体：
   - 总体要求或工作目标。
   - 主要任务。
   - 工作要求。
5. 落款和日期。

会议通知：

1. 标题：`关于召开XX会议的通知`。
2. 首段：目的 + 决定召开会议。
3. 主体：会议主题、时间、地点、参会人员、议程、有关要求、联系人。

印发通知：

1. 标题：`关于印发XX的通知`。
2. 首段：起因目的 + 依据 + `现将《XX》印发给你们，请结合实际抓好落实`。
3. 附件或正文。

## 工作方案

适用：具体部署某项未来工作。

完整型：

1. 标题：`XX工作方案`。
2. 首段：背景 + 目的 + 依据 + `制定本方案`。
3. 一、总体要求：指导思想、基本原则、工作目标。
4. 二、主要任务。
5. 三、工作要求：组织领导、责任分工、协调保障、督导考核、宣传引导。

直接型：

1. 标题和首段。
2. 对外措施。
3. 对外措施。
4. 对内保障措施。

规则：

- 不能照搬上级方案。
- 重点是把上级要求转成本级能执行的任务、责任和步骤。

## 述职报告

适用：个人或领导履职情况报告。

核心规则：上级通知就是框架。不得脱离通知要求随意写单位总结。

流程：

1. 提取上级要求述职的方向。
2. 按方向匹配领导分管领域或个人岗位成绩。
3. 用具体工作和数据支撑。
4. 先定框架，再展开。

## 调研报告

适用：围绕一个窄问题或经验开展研究。

经验型：

1. 标题：`关于XX的调研报告`。
2. 首段：调研背景 + 调研过程 + 引出下文。
3. 一、主要情况。
4. 二、主要工作做法。
5. 三、经验及建议。

问题型：

1. 首段。
2. 一、主要情况。
3. 二、存在的主要问题。
4. 三、对策及建议。

规则：

- 选题越窄越好。
- 数据要比普通总结更充分。
- 对策建议是建议，不是已经决定的工作。

## 理论文章

适用：理论阐释、署名文章、思想性文章。

结构接近事前类领导讲话：

1. 标题。
2. 首段。
3. 第一部分：背景、意义、形势、成绩。
4. 第二部分：需要做的核心工作。
5. 第三部分：保障工作。

语言要求：比讲话更书面、更权威、更少口语。

## 新闻稿

适用：近期会议、活动、调研新闻。核心是说明发生了什么。

共同要求：

- 标题直接说明事件。
- 首段写时间、地点、人物、事件四要素。

会议新闻：

1. 标题：某单位召开某会议。
2. 首段：四要素。
3. 主体：会议背景或议程；领导讲话详情。
4. 常用词：会议指出、会议认为、会议明确、会议强调、会议要求。

活动新闻：

1. 标题：某单位开展某活动。
2. 首段：四要素。
3. 主体：活动经过；活动影响。

调研新闻：

1. 标题：某领导到某地调研。
2. 首段：四要素，突出主要领导。
3. 主体：调研过程；座谈会议过程；领导肯定和要求。

## 通讯稿

适用：面向公众宣传单位工作成绩或先进人物。

结构：

1. 标题：实标题 + 虚标题，可二者结合。
2. 开头：小故事或小场景，再引出整体工作和数据。
3. 主体 3-5 个分点：
   - 故事或场景。
   - 问题或冲突。
   - 措施办法。
   - 解决结果。

规则：

- 以小见大。
- 段落短、画面多。
- 冲突来自真实困难，不编造故事。

## 先进事迹

适用：推荐、表彰或宣传个人。

内部简报式：

1. 标题：`XX同志先进事迹`。
2. 首段：姓名、出生年月、政治面貌、现任职务、主要成绩、获奖情况。
3. 主体：围绕表彰方向写本职工作成绩。

规则：

- 不写空泛美德。
- 用岗位成果和数字证明先进。
- 先看表彰通知，不能夸错方向。

## 民主生活会对照检查材料

适用：按照上级要求查摆问题、剖析原因、提出整改。

结构：

1. 标题。
2. 首段：会前准备情况。
3. 一、存在的主要问题：严格对应上级要求的若干方面。
4. 二、原因剖析。
5. 三、下一步整改措施：与问题一一对应。

规则：

- 不能把成绩当主体。
- 上级列几个方面，就对应几个方面。
- 问题和整改不能脱节。

## 党课

适用：党性教育、理论学习、专题授课。

框架一：

1. 首段：党课背景 + 主题。
2. 一、思想意义层面：水分成分 + 工作要求。
3. 二、核心工作：水分成分 + 工作措施。
4. 三、对内要求：水分成分 + 工作措施。
5. 结尾：号召。

框架二：

1. 按三个措施主题展开。
2. 每个主题内部结合水分成分和工作措施。

## 微型党课 / 演讲

适用：短篇党课、演讲比赛、主题宣讲。

结构：

1. 故事或场景开头。
2. 点出主题。
3. 引申到历史、理论、引用或意义。
4. 收回到当前工作或个人岗位。
5. 分解为 2-3 个行动要求。
6. 结尾号召并回扣主题。

## 会议纪要

适用：会后形成正式摘要，不是会议记录。

首要规则：优先沿用本单位历史纪要格式。

一事一议型：

1. 标题。
2. 首段。
3. `(一) 审议通过A事项`：情况、决定、下一步安排。
4. `(二) 研究讨论B事项`：情况、决定、下一步安排。

会议新闻型：

1. 标题。
2. 首段。
3. 会议指出。
4. 会议明确。
5. 会议要求。
6. 会议强调。

取舍规则：纪要只保留会议决定、核心意见和主要领导最终意见，不写流水账。
