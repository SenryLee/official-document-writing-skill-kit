# 成分、逻辑与高度处理

用于搭段落、诊断结构、提升文章高度、处理产品化表达和风格转换。

## 目录

- 公文成分
- 水分与干货
- 因果结构
- 六大逻辑
- 地空对接
- 产品化
- 概念封装
- 书面语和口语化

## 1. 公文成分

公文不是从第一个字写到最后一个字，而是由不同成分拼装而成。

| 成分 | 作用 | 常见场景 |
|---|---|---|
| 意义 | 说明工作为什么重要 | 讲话、理论文章、党课、心得 |
| 背景 | 客观交代时间、形势、事件、政策 | 首段、报告、新闻、讲话 |
| 形势 | 同时写有利条件和压力挑战 | 报告、讲话、计划、调研 |
| 问题 | 写不足、短板、风险、困难 | 总结、调研、对照检查 |
| 解释 | 解释概念、政策、原因 | 讲话、培训、理论文章 |
| 成绩 | 写已经完成的成果 | 简报、总结、报告、述职 |
| 引用 | 借助上级文件、领导讲话、经典表述提升权威 | 讲话、理论文章、党课 |
| 例子 | 用具体案例证明观点 | 讲话、通讯稿、调研 |
| 故事 | 提升可读性和感染力 | 通讯稿、微型党课、演讲 |
| 假设 | 写不做某事的后果 | 讲话、论证段落 |
| 正反论证 | 从正反两面强化观点 | 讲话、理论文章、调研 |
| 要求 | 提出方向性要求 | 通知、讲话、方案 |
| 措施 | 说明具体怎么做 | 几乎所有文种 |
| 计划 | 说明未来怎么做 | 计划、总结、报告、方案 |
| 目标 | 说明预期状态和指标 | 计划、报告、方案 |
| 号召 | 收束全文、动员行动 | 讲话、党课、报告 |

## 2. 水分与干货

干货是高密度工作信息：

- 措施
- 计划
- 目标
- 工作总结、报告中的成绩

水分帮助读者理解、接受和进入语境：

- 意义、背景、形势、解释、问题、引用、例子、故事、假设、正反论证、号召

领导、专家、内部汇报、总结、报告、方案、通知：多用干货。

讲话、宣传、通讯稿、党课、培训材料：适度增加水分。

## 3. 因果结构

复杂文章常用：

`因：意义/背景/形势/问题/引用/例子/故事/假设/正反论证 -> 果：要求/措施/计划/目标/成绩/号召`

写讲话和理论文章时，不要只有“为什么重要”，最后必须落到“怎么做”。

## 4. 六大逻辑

| 逻辑 | 含义 | 适用 |
|---|---|---|
| 从古到今 | 按时间先后 | 过程、背景、成绩、新闻、故事 |
| 从上到下 | 上级来源到本级落实 | 政治性任务、政策贯彻、领导责任 |
| 从大到小 | 范围或重要性由大到小 | 总结、报告、计划、讲话 |
| 从虚到实 | 思想、原则、认识到具体行动 | 党建、理论、讲话、心得、方案 |
| 从外到内 | 外部环境、服务对象到内部保障 | 调研、服务工作、讲话 |
| 从知到行 | 认识到行动 | 心得、培训、党课、动员讲话 |

不要硬套所有逻辑。选择最能让材料清楚的一种作为主逻辑即可。

## 5. 地空对接

当文章低端、像流水账、像杂活清单时，使用地空对接。

方法：

1. 找到地面工作：开会、打扫、服务、培训、填表、检查、后勤等。
2. 找到真实对应的上级精神、年度重点任务、单位战略或领导要求。
3. 把上级精神放在句首或段首。
4. 把基层工作落到更大的目标上。

安全骨架：

`围绕[真实上级要求/重点任务]，开展[本级具体工作]，推动[上级目标/本单位目标]。`

规则：

- 上级提法必须真实、准确、原文使用。
- 上级精神放在句子或段落开头。
- 只链接相关精神，不生拉硬扯。
- 段首已经完成链接后，后续分点不必句句重复。

## 6. 产品化

当普通工作需要形成亮点时，使用产品化。

逻辑：

`上级精神 -> 中间层产品/工程/行动/机制 -> 基层具体工作`

常用产品词：

- 工程
- 行动
- 机制
- 平台
- 专班
- 体系
- 清单
- 方案
- 模式
- 品牌
- 阵地

规则：

- 只能在中间层创造名称。
- 上级已经明确的名称不能改。
- 单位领导常用的说法不能改。
- 产品名必须能拆解出真实工作内容。

## 7. 概念封装

用于把多项工作压缩成一个好识别的概念。

常见形式：

- `"1+N"机制`
- `"三单闭环"工作法`
- `"五个一"行动`
- `"党建+服务"平台`

一个合格概念通常包含：

- 一个醒目的双引号概念。
- 一个务实名词：机制、平台、行动、工程、模式、清单。
- 下文能解释的真实内容。

不要制造解释不了的漂亮词。

## 8. 书面语和口语化

口头语太重时：

- 删除闲聊式过渡、故事、重复解释、个人感受。
- 改成措施、结果、要求和清晰标题。

讲话太书面时：

- 增加简短例子、故事、对比或号召。
- 段落更短。
- 最后仍落到具体工作。
