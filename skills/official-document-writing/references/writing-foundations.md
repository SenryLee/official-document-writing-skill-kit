# 公文基础写法

用于处理标题、句式、首段和公文化语言。

## 目录

- `V+N`：措施和计划
- `N+V`：成绩和结果
- `3V+N`：完整工作句
- 目标、过程、措施、结果
- 首段 `按-拿-推`
- 标题结构
- 书面语控制
- 数据和事实控制

## 1. `V+N`：措施和计划

`V+N` 是“动词 + 工作名词”。它主要表达措施、办法、任务和未来安排。

常见形式：

- 强化组织领导
- 完善制度机制
- 开展专项整治
- 推进项目建设
- 做好服务保障

使用规则：

- 尽量以动词开头。
- 标题或首段已经说明主体时，后文不反复写单位名称。
- 逗号前的短句尽量精炼；超过 25 个字要检查是否冗长。
- 能删掉的“的”尽量删掉。
- 写未来计划时，优先使用 `V+N`，不要写成已经完成的结果口吻。

## 2. `N+V`：成绩和结果

`N+V` 是“工作名词 + 结果动词/状态词”。它主要表达已经完成的成绩、状态和结果。

常见形式：

- 项目建设稳步推进
- 服务效能持续提升
- 风险隐患有效化解
- 工作机制更加健全

使用规则：

- 主要用于总结、报告、简报标题和成绩句。
- 不适合作为未来计划的主体句式。
- 最大成绩、最终结果、最宏观成果放前面。
- 能配数据时必须配数据。

## 3. `3V+N`：完整工作句

当一句话要完整表达一项工作时，使用：

`工作名称/来源 -> 工作方式 -> 工作内容 -> 工作成绩`

可按五个维度组织：

1. 工作名称：这项工作叫什么。
2. 工作来源：来自哪个上级部署或单位安排。
3. 工作方式：通过什么方式开展。
4. 工作内容：具体做了什么。
5. 工作成绩：形成什么结果、数据或状态。

不要机械凑满三个动词。材料需要简洁时，一个动词也可以；需要丰富时，再扩展为多个动作。

## 4. 目标、过程、措施、结果

较完整的公文句子可分为四部分：

| 部分 | 作用 | 常见词 |
|---|---|---|
| 目标 | 方向、原则、战略 | 按照、围绕、坚持、以……为 |
| 过程 | 路径、方法组合 | 通过、采取、依托、借助 |
| 措施 | 具体动作 | 开展、推进、完善、建立 |
| 结果 | 成绩、状态 | 实现、形成、提升、取得 |

大型总结、报告经常只写“措施 + 结果”；工作简报、讲话、宣传稿可以适当写完整一些。

## 5. 首段 `按-拿-推`

多数正式材料和章节首段可用 `按-拿-推`：

- **按**：上级来源、政治方向、战略要求、政策依据。
- **拿**：主要抓手、重点措施、总体做法，通常 3-5 项。
- **推**：最终成绩、预期目标或工作状态。

通用骨架：

`[时间状语]，[单位]深入贯彻落实[上级精神/会议部署]，围绕[核心任务]，坚持[措施一]、[措施二]、[措施三]，推动[工作目标/工作成效]。`

通知和方案的首段可以更直接：

`为[目的]，根据[依据]，结合[实际]，现就有关事项通知如下。`

## 6. 标题结构

| 类型 | 结构 | 适用 |
|---|---|---|
| 独角兽型 | `V+N` 或 `N+V` | 正式行文、低层级标题、直接任务 |
| 双子座型 | `V+N，V+N` 或 `V+N，N+V` | 一级标题、讲话、总结、报告 |

标题质量要求：

- 同级标题结构尽量一致。
- 大标题要统领小标题，小标题要回应大标题。
- 不写过大的空标题，主题越窄越容易写实。
- 概念封装只能概括真实工作，不能只为好看。

## 7. 书面语控制

总结、报告、通知、方案、纪要应偏书面化：

- 多写措施、计划、目标、成绩。
- 少写故事、对话、感受和反复解释。

讲话、党课、演讲、通讯稿可以适度口语化：

- 可以加例子、故事、引用、对比。
- 最后仍要落到工作要求或行动。

## 8. 数据和事实控制

- 缺少数字时写 `[待补充数据]`。
- 不编造、不美化、不四舍五入事实。
- 成绩模糊时，优先追问数量、频次、覆盖范围、金额、比例、奖项、解决问题数。
- 文件、会议、政策、机构、领导名称必须按用户提供的原文使用。
