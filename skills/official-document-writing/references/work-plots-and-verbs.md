# 工作剧情与常用动词

用于在素材较少时寻找工作表达维度。剧情只能帮助组织已知事实，不能用来编造事实。

## 目录

- 四个工作阶段
- 通用句子公式
- 工作启动阶段
- 工作执行阶段
- 工作保障阶段
- 工作反馈阶段
- 常用动词表

## 1. 四个工作阶段

| 阶段 | 作用 | 常见内容 |
|---|---|---|
| 启动阶段 | 把想法变成组织化工作 | 学习、调研、会议、方案、制度、领导小组、责任分工 |
| 执行阶段 | 把纸面要求变成实际行动 | 活动、载体、项目、平台、服务、专项行动 |
| 保障阶段 | 为工作推进提供支撑 | 政策、资金、人才、资源、后勤、协调 |
| 反馈阶段 | 形成闭环 | 督导、整改、考核、宣传、总结 |

稿件空泛时，检查是否缺少某一阶段的事实。

## 2. 通用句子公式

多数工作剧情可以按这个公式写：

`工作名称 + 工作形式 + 工作内容 + 成果/数据`

常用骨架：

`开展[专项工作]，通过[形式一]、[形式二]等方式，围绕[内容]进行[动作]，累计[数据]，形成[成果]。`

`围绕[上级要求/核心任务]，建立[机制/平台/专班]，明确[责任/流程/标准]，推动[结果]。`

## 3. 工作启动阶段

### 调查调研

用于了解情况、发现问题、摸清底数。

- 维度：调研名称、调研对象、调研方式、调研内容、调研次数、问卷数量、座谈次数、形成成果。
- 动词：开展、组织、走访、调研、收集、梳理、研判、形成。
- 骨架：`开展[主题]专项调研，通过[问卷/座谈/走访]等方式，围绕[重点内容]收集意见建议，梳理形成[问题/清单/报告]。`

### 会议研究

用于研究、决策、部署、协调。

- 维度：会议名称、会议任务、参会人员、议题、频次、议程、决定事项。
- 动词：组织、召开、召集、研究、部署、明确、协调、推动。
- 骨架：`召开[会议名称]，组织[参会人员]围绕[议题]开展研究，明确[任务/责任/时限]。`

### 学习贯彻

用于政治、理论、政策、纪律、意识形态等务虚内容。

- 维度：学习名称、学习内容、学习对象、学习方式、学习次数、学习成效。
- 动词：学习、贯彻、领会、把握、紧扣、坚持、研究。
- 骨架：`深入学习贯彻[文件/会议/讲话精神]，通过[中心组学习/专题培训/交流研讨]等方式，引导[对象]准确把握[重点内容]。`

### 制度机制

用于规则、流程、标准和长效运行。

- 维度：制度名称、制度体系、制度内容、流程标准、功能作用、执行情况。
- 动词：制定、建立、出台、健全、完善、优化、规范、实施、推行、落实。
- 骨架：`建立健全[制度机制]，围绕[重点环节]细化[标准/流程/责任]，推动[工作]规范运行。`

### 组织领导与责任

用于领导小组、专班、责任分工、协调机制。

- 维度：领导小组、工作专班、部门职责、责任人、协调机制、考核机制。
- 动词：成立、设立、明确、细化、统筹、协同、压实、加强、抓好。
- 骨架：`成立[机构/专班]，明确[部门/人员]职责，构建[协调/推进/督导]机制，压实工作责任。`

## 4. 工作执行阶段

### 活动载体

用于把政策、精神、要求落到具体行动。

- 维度：活动名称、载体名称、服务对象、活动内容、频次、覆盖范围、结果。
- 动词：开展、打造、推进、组织、实施、搭建、服务、促进。
- 骨架：`依托[平台/载体]，开展[活动/行动]，围绕[内容]服务[对象]，推动[目标]落地见效。`

### 项目与平台

用于工程、数字系统、窗口服务、品牌、示范点。

- 维度：项目名称、平台功能、建设内容、服务对象、投入资源、运行成效。
- 动词：建设、打造、升级、接入、运行、推广、赋能。
- 骨架：`建设[平台/项目]，完善[功能模块]，服务[对象/场景]，提升[效率/质量/覆盖面]。`

## 5. 工作保障阶段

### 政策保障

- 生命周期：制定政策、学习政策、落实用好政策、争取政策。
- 动词：制定、学习、落实、用好、争取、对接、承接、转化。
- 骨架：`积极对接[政策资源]，研究吃透[政策内容]，推动[政策]转化为[项目/资金/服务/机制]。`

### 资金保障

- 维度：资金来源、争取金额、使用方向、管理方式、监督机制、效益。
- 动词：募集、争取、引进、整合、使用、管理、监督、监管。
- 骨架：`统筹整合[资金来源]，重点保障[工作方向]，强化资金使用监管，提升资金使用效益。`

### 人才保障

- 生命周期：引进人才、培养人才、使用人才、激励人才。
- 动词：引进、培育、培训、邀请、使用、激励、管理、帮助。
- 骨架：`实施[人才计划]，通过[引育用留措施]，为[工作]提供人才支撑。`

## 6. 工作反馈阶段

### 督导督促

- 维度：对象、内容、方式、频次、问题、整改要求。
- 动词：督导、检查、巡视、排查、查漏、反馈、推动。
- 骨架：`建立[督导机制]，围绕[重点任务]开展检查，及时反馈问题并推动整改落实。`

### 问题整改

- 维度：问题来源、问题清单、责任分工、整改措施、整改时限、回头看。
- 动词：查摆、梳理、建账、整改、销号、回访、巩固。
- 骨架：`针对[问题]建立整改台账，明确[责任/措施/时限]，实行闭环管理。`

### 检查考核

- 维度：指标、标准、责任、结果运用、奖惩。
- 动词：考核、考评、检验、纳入、奖励、问责、总结。
- 骨架：`将[工作]纳入[考核体系]，细化[指标标准]，强化结果运用。`

### 典型宣传

- 维度：宣传方式、宣传内容、宣传次数、宣传范围、宣传层次、影响。
- 动词：宣传、推介、发布、报道、总结、提炼、营造。
- 骨架：`通过[媒体/平台/活动]宣传[内容]，扩大[工作]覆盖面和影响力，营造[氛围]。`

## 7. 常用动词表

| 类别 | 动词 |
|---|---|
| 目标方向 | 围绕、聚焦、坚持、立足、着眼、紧扣 |
| 学习贯彻 | 学习、贯彻、领会、把握、研究、落实 |
| 会议组织 | 组织、召开、召集、举行、部署、研究 |
| 制度机制 | 制定、建立、健全、完善、优化、规范、推行 |
| 执行推进 | 开展、推进、实施、打造、搭建、深化、提升 |
| 责任落实 | 明确、细化、压实、统筹、协同、强化 |
| 保障支撑 | 争取、整合、引进、配套、保障、支撑 |
| 反馈闭环 | 督导、检查、整改、考核、问责、总结 |
| 结果表达 | 取得、形成、实现、建成、提升、确保、营造 |
